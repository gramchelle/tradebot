# TradeBot

TradeBot is an AI implementation Spring-Boot backend app for analyzing trends in stock market.

## Technical Stack

* Java 21
* Spring-Boot Framework
* Python 3.12
* GraalVM (TBD)
* Apache Kafka (TBD)

## Layers

1. Technical Analysis
2. Statistical Analysis
3. AI, ML based algorithms
4. Risk Management
